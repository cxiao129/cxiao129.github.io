了解PHP

什么是PHP？

  • PHP 脚本在服务器上执行
  • PHP 文件能够包含文本、HTML、CSS 以及 PHP 代码
  • PHP 代码在服务器上执行，而结果以纯文本返回浏览器
  • PHP 文件通常包含 HTML 标签以及一些 PHP 脚本代码
  • PHP 脚本可放置于文档中的任何位置

PHP 能够做什么？

  • PHP 能够生成动态页面内容
  • PHP 能够创建、打开、读取、写入、删除以及关闭服务器上的文件
  • PHP 能够接收表单数据
  • PHP 能够发送并取回 cookies
  • PHP 能够添加、删除、修改数据库中的数据
  • PHP 能够限制用户访问网站中的某些页面
  • PHP 能够对数据进行加密

PHP快速入门

PHP变量

变量是存储信息的容器

在 PHP 中，所有用户定义的函数、类和关键词（例如 if、else、echo 等等）都对大小写不敏感；但是所有的变量都对大小写敏感。

PHP是一种弱类型语言，或者说是动态类型语言，变量会在首次为其赋值时被创建。

变量以 $ 符号开头，其后是变量的名称。

变量的作用域

PHP 有三种不同的变量作用域：

  1. local（局部）
  2. global（全局）
  3. static（静态）

local和 global 作用域

函数之外声明的变量拥有 global 作用域，只能在函数以外进行访问，在函数内部不可见。

函数内部声明的变量拥有 local 作用域，只能在函数内部进行访问，在函数外部不可见，函数执行完毕后被删除。

PHP global 关键词

global 关键词用于在函数内部访问全局变量，从而在函数内部也可以调用和修改全局变量（是变量本身，而非副本）。

要做到这一点，请在函数内部的变量名前面使用 global 关键词，名称要与全局变量名称保持一致。

PHP static 关键词

通常，当函数执行后，会删除所有变量。不过，有时需要保留某个局部变量的值。

要实现这一点，要在首次声明变量时使用 static关键词，然后每当函数被调用时，这个变量所存储的信息都是函数最后一次被调用时所包含的信息，而不会重新被赋初值。

注意：该变量仍然是函数的局部变量。

单双引号和echo

  • echo - 能够输出一个以上的字符串，例如 

     echo "This", " string", " was", " made", " with multiple parameters.";

  • print - 只能输出一个字符串，并始终返回 1

注意：echo 比 print 稍快，因为它不返回任何值。echo 是一个语言结构，有无括号均可使用：echo 或 echo()。

将简单变量写入一个由双引号引用的字符串中，就叫插补操作。例如： 

 $test = "xx";
 echo "你好：$test"

PHP数据类型

var_dump() 函数用于输出变量的相关信息。

var_dump() 函数显示关于一个或多个表达式的结构信息，包括表达式的类型与值。数组将递归展开值，通过缩进显示其结构。

<?php 
$a = array(1, 2, array("a", "b", "c")); 
var_dump($a); 
?>

输出结果为

array(3) {
  [0]=>
  int(1)
  [1]=>
  int(2)
  [2]=>
  array(3) {
    [0]=>
    string(1) "a"
    [1]=>
    string(1) "b"
    [2]=>
    string(1) "c"
  }
}
